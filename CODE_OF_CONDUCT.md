# 社区行为准则

## 我们的承诺

我们致力于为所有参与者提供开放、友善、无骚扰的协作环境，不因年龄、身体特征、残障、族裔、性别认同与表达、经验水平、教育程度、国籍、外貌、种族、宗教、性取向或社会经济状况而区别对待。

## 鼓励的行为

- 以事实、复现步骤和代码证据讨论技术问题。
- 尊重不同经验背景，接受建设性反馈。
- 对安全、隐私和上游权利边界保持谨慎。
- 关注对社区和使用者最有利的结果。

## 不可接受的行为

- 骚扰、侮辱、威胁、歧视或人身攻击。
- 未经许可公开他人的私人信息或凭据。
- 故意提交恶意代码、真实秘密或来源不明的上游代码。
- 持续扰乱 issue、PR、讨论区或其他项目空间。

## 执行

维护者有权删除、编辑或拒绝违反本准则的评论、提交、issue 和其他贡献，并可临时或永久限制参与。需要私下报告行为问题时，请通过维护者 GitHub 主页列出的联系渠道沟通；维护者将尽力保护报告者隐私。

本准则适用于仓库、issue、PR、讨论区以及代表本项目参与公共活动的场合。内容参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)，原作版权属于 Contributor Covenant 贡献者，并按 [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) 使用；本文按本项目规模进行了简化。
